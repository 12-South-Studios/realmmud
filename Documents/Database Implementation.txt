DatabaseClient Functions
BeginTransaction (No Params)
- Increments local transaction id, creates new PendingTransaction

AddDbCommand (Command Name, Parameter Dictionary, Callback Function, Temp Data Dictionary)
- Creates new PendingCommand
- Adds cmd to the Current Transaction

PerformTransaction (Callback Function, Temp Data Dictionary)
- Sends the set of db commands to a random database object

OnTransactionComplete (Event)
- Callback Function from the databsae call


PendingTransaction Object
- DatabaseClient
- TransactionId
- Temp Data Dictionary
- Callback Function
- Pending Command List

PendingCommand Object
- DatabaseClient
- Callback Function
- Temp Data dictionary
- Stored Procedure


DatabaseServer Object
- Maintains a connection to the database
- Executes based upon event calls so it is threaded
- Performs rollback or commit of transaction
- Issues callbacks
